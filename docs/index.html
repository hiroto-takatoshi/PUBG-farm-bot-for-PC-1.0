💾 Latest version | 最新版 

      This bot currently only works for 1920*1080, full screen(windowed), display language English. 
      For other resolution, you need to adjust parameters by yourself. 

      Servers: NA:1|EU:2|KRJP:3|AS:4|OC:5|SA:6|SEA:7  
      Modes: SOLO:1|DUO:2|SQUAD:3 

      Example: If you want to farm for NA-Solo, set config.json as "11", 
      if you want to bot for AS-DUO, set as "42".  

      If you want to farm for BP, set waittime as "220". 
      If you only want to lower your rank, set waittime as "0". 

      Attention: Must set and only set 5 models. 
      The probability to choose one certain model is same for the five models.

      - 语言调至英文
      - 分辨率设为1920*1080，全屏窗口化
      - 配置文件config.json格式见下方
